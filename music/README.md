Drop your birthday music MP3 file in this folder!

## Quick Setup:

1. Add your MP3 file here (e.g., `birthday-song.mp3`)
2. Update the path in `src/app/components/finale/finale.ts` (line 64):
   ```typescript
   this.audio.src = 'music/your-filename.mp3';
   ```

## Recommended:

- File format: MP3
- File size: Under 5MB
- Length: 2-3 minutes
- Use royalty-free or licensed music

See `MUSIC_GUIDE.md` in the project root for detailed instructions!
